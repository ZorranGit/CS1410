Exercise: get_dict_value02

Description

In this exercise, your function will receive 2 parameters, a dictionary, and string that is a key in the dictionary. The function will find and return the value associated with the key.

Function Name

get_dict_value02

Parameters

  • dictionary : a dictionary
  • key : a key in dictionary

Return Value

The value associated with key in dictionary.

Examples

get_dict_value02({ "a": 1}, "a") -> 1
